# Lorem Ipsum Totalis

Generate themed placeholder text from any word or phrase.

![Screenshot](images/screenshot.png)

## Features

- Turborepo
- Typescript
- Vite
- Vue 3
- Tailwind CSS
- Eslint
- Prettier
- Lint-staged
- Emoji conventional commits

## Installation

```bash
yarn install
```

## Development

```bash
yarn dev
```

To commit changes use the commit helper script to build a valid conventional commit

```bash
yarn commit
```
